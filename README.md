Overview: This app will tell the user the time for each prayer time based upon the user's current location.

Features:
  1. Push Notifications (Includes: Reminders to pray, the Athan, etc) via the FCM Library.
  2. Simple user customization that gets stored properly and safely in the user's device (Includes: Language, the Muathans voice, Light/Dark Mode) with ROOM.
  3. Gathering Athan timing via the Athan API with Retrofit and accurately presenting their cities data via gianing user access of their location.
  4. Accurately present the user a Qibla finder compass that changes based upon the users location and direction.
  6. Using ViewModel and NavHost to make allow for a smooth experience while maintaining the readability and a clean connection between the UI and View layers of the codebase.
